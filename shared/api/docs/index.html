A simple REST API for searching and listing trading game cards

Cards

Search and favorite trading game cards

List Cards

Search for, and list trading game cards that match a search term

query Parameters
term
required
string (SearchTermParam) [ 3 .. 50 ] characters
Example: term=pokey

The search term to find cards with

orderby
string (OrderByParam)
Default: "name"
Enum: "name" "released"
Example: orderby=name

On what property to order the matching cards. Default is name

sortdir
string (SortDirParam)
Default: "AUTO"
Enum: "ASC" "DESC" "AUTO"
Example: sortdir=AUTO

The sort order of the returned cards. Default is AUTO Sort order:

  • ASC - Ascending, from A to Z
  • DESC - Descending, from Z to A
  • AUTO - Allow the system choose the most inuitive direction to sort.
page
integer (PageParam) [ 1 .. 50 ]
Default: 1
Example: page=1

The page of the matching resultset to return. Default is 1.

Responses

http://127.0.0.1:8889/search
